# Number-Guessing-Game
A simple command-line number guessing game written in Python. The player selects a difficulty level, and the program generates a random number within a corresponding range. After each guess, the game provides feedback until the correct number is guessed.

## Features

- Five difficulty levels
- Random number generation using Python's `random` module
- Unlimited guessing attempts
- Hints after each incorrect guess ("Too high" or "Too low")
- Tracks the total number of attempts

## Difficulty Levels

| Difficulty | Range |
|------------|-------|
| 0 - Baby | 0 - 2 |
| 1 - Easy | 1 - 5 |
| 2 - Normal | 1 - 10 |
| 3 - Hard | 1 - 100 |
| 4 - Very Hard | 1 - 1000 |

## Technologies Used

- Functions
- Loops
